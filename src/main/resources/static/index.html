Описание модуля "Освещение"

Данный модуль управляет освещением зданий. Реализовано оперативное включение и отключение, работа по расписанию. Для работы по расписанию необходимо зарегистрировать в системе пользователя, устройство управления нагрузкой, назначить устройству интерфейс для управления про протололу SNMPv1

Описание методов модуля щсвещение

  • http://sarov.us.to:9900/snmp/on включение освещения тестового объекта
  • http://sarov.us.to:9900/snmp/off отключение освещения тестового объекта
  • http://sarov.us.to:9900/snmp/state состояние силовой цепи освещения тестового объекта
  • http://sarov.us.to:9900/snmp/scheduler/users/add - ввод нового пользователя. GET queryparam. В параметре username=Ivan Ivanov передать уникальное имя нового пользователя
  • http://sarov.us.to:9900/snmp/scheduler/device/add - ввод нового устройства. GET queryparam. community=public&ip=192.168.1.1&name=ERD1. Имя устройства и Ip адрес должны быть уникальными. Community - параметр для SNMP v1
  • http://sarov.us.to:9900/snmp/scheduler/job/add - ввод новой задачи. GET queryparam. job_ts=1540148060&user=Ivan Ivanov&command=1&device_name=ERD1. job_ts - время выполнения задачи в формате UNIX timestamp. Command 1/2 для тестового объекта включить/отключить соответственно
  • http://sarov.us.to:9900/snmp/scheduler/job/show - отобразить расписание. GET queryparam. min_ts=0&max_ts=1600000000. min_ts, max_ts - временной интервал в формате UNIX timestamp. Возвращает json с описание задач. id - индекс задания. done - отметка о выполнении
  • http://sarov.us.to:9900/snmp/scheduler/job/remove - удалить задание. GET queryparam. id=16. id - индекс удаляемой задачи
